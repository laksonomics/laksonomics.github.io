# Riandy Laksono Academic Website

Live site: https://laksonomics.github.io/

## Structure

- `index.html`: academic homepage (profile, publications, commentary, media, blog entry points)
- `blog/index.html`: blog landing page
- `blog/us-indonesia-art-explainer.html`: US-Indonesia ART interactive explainer
- `US_IDN_ART_baby_explainer_detailed.html`: detailed ART explainer
- `reproducible-code/index.html`: reproducible-code hub with per-publication workspaces
- `assets/css/styles.css`: site-wide styling
- `assets/images/riandy-laksono-profile.jpg`: profile photo
- `assets/docs/CV_RL_2026.pdf`: CV source used for profile and publication updates

## Notes

- Visual direction adapted from: https://github.com/sbryngelson/academic-website-template
- Publication cards include both `Paper` and `Code` buttons.
- Publication links are set to exact publisher pages sourced from `CV_RL_2026.pdf` and publisher/DOI pages.
- ART content is nested under the Blog section, while the root page is the academic profile.
